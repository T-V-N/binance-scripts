# binance-scripts

**HOW TO**
1. Run 
>npm install
2. Create .env file inside root directory with the following structure:
  >API_KEY=INSERT KEY HERE  
  >API_SECRET=INSERT KEY HERE  
  >MARKETS="BTCUSDT, BNBUSDT, LTCUSDT, ETHUSDT, DOTUSDT"  
  >HOST_URL=https://fapi.binance.com  
  
3. Run
>npm start
